# 프라임칼리지 R 컴퓨팅 수시과제2

## 패키지 설치

`problem2.R`에서는 `MASS` 패키지의 `Cars93` 데이터를 사용합니다.
과제를 실행하기 전에 R 콘솔에서 다음 명령을 한 번 실행해 패키지를 설치합니다.

```r
install.packages("MASS", repos = "https://cran.r-project.org")
```

이미 설치되어 있다면 설치 명령은 다시 실행하지 않아도 됩니다.
스크립트를 실행할 때는 다음처럼 패키지를 불러옵니다.

```r
library(MASS)
```

설치가 제대로 되었는지 확인하려면 R 콘솔에서 아래 명령을 실행합니다.

```r
head(Cars93)
```

`Cars93` 데이터 일부가 출력되면 정상적으로 준비된 상태입니다.
