질적 qualitative
양적 quantitative

질적변수 종류
nominal 명목형
ordinal 순서형

양적변수 종류
continuous 연속형
discrete 이산형

질적데이터는
막대그래프 쓰면 좋음
원그래프는 잘 읽기 어려워서 (예를들어 두 항목중 뭐가 더 큰지) 안좋음

양적데이터는
히스토그램 쓰면 좋음
히스토그램은 한눈에 파악하기 좋고 특이점(outlier)이 잘보임
대신 구간을 어떻게 잡냐에 따라 해석이 달라지기 쉬움

데이터가 작으면 점도표를 써볼 수 있음

mode 최빈값
mean 평균

데이터의 무게중심이 평균임
= 관찰값 전체의 합 / 관찰값 갯수

기울어진(skewed) 분포를 가졌거나 outlier가 있으면 데이터를 잘 대표하지 못함
좌우대칭 데이터면 잘 대표함

편차: 관찰값 - 평균
variance 분산: 편차 제곱의 평균

편차를 평균내면 항상 0이라 의미x
그래서 제곱하고 평균을 씀 그게 분산

표본분산 s^2 = sum(편차^2) / (n-1)
n으로 나누는 것보다 n-1로 나누는게 모분산에 더 가까워서 n-1을 씀

표본분산은 모분산을 추정하기 위해 씀
모분산은 내가 놓쳤거나 강의에서 안 알려준거 같은데 아마 진짜(?)분산을 말하는거 같음

표준편차: 분산은 제곱이 걸려있어서 스케일이 커졌으니 그걸 다시 루트씌워놓은것
= root(s^2)
